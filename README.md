# Abalo, Joshua
While building the core features of the HR Applicant Window System, I used AI tools like Gemini and GitHub Copilot to act as virtual coding partners. They were incredibly helpful when I was mapping out the backend logic, such as figuring out how applicant profiles are processed and how their statuses are tracked as they move through the hiring pipeline. Getting the HR system to communicate securely with the database—ensuring that sensitive applicant information and records are stored and retrieved safely—can be tricky. The AI provided clear syntax examples and step-by-step configuration help to get the connection running properly. Whenever I ran into a connection error or a bug while trying to pull an applicant's record, I could review the error logs with the AI to quickly track down the exact fix. This saved me hours of frustration and ensured the system's data flows smoothly and reliably.
